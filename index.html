Kết nối Google Sheet qua Apps Script

Mặc định dashboard chạy với dữ liệu mẫu để demo giao diện. Để lấy dữ liệu thật:
1. Publish Apps Script (Code.gs đi kèm) dạng Web App, quyền Anyone.
2. Dán URL /exec vào ô trên, hoặc điền cố định vào hằng số APPS_SCRIPT_URL ở đầu file trước khi đưa lên GitHub Pages.
3. URL sẽ được lưu trong phiên làm việc hiện tại (không lưu trình duyệt) — dữ liệu thật luôn lấy từ Google Sheet mỗi khi tải trang hoặc bấm "Đồng bộ".
